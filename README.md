## Setup
1. Clone:
   ```bash
   git clone git@github.com:divyanshdj/olympics-web.git
   cd olympics-web
   ```
2. Install:
   ```bash
   npm install
   ```
3. Run:
   ```bash
   npm start
   ```
   Open `http://localhost:5173`.

4. Build:
   ```bash
   npm run build
   ```
